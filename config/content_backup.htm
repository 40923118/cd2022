學長推甄與考研的相關心得

cadlab startup

  1. 利用批次檔案關閉 IPv4, Proxy 自動偵測, 自行設定 IPv6 DNS 與 Proxy.
  2. 利用 Hybrid 啟動可攜系統.
  3. 利用 .reg 設定 Putty session.
  4. 啟動可攜系統時同時開啟位於 F: 中的 NX2008

NXOpen Python

  1. 選擇將 NXOpen Journal 檔案儲存格式設為 Python. File -> Preferences -> User Interface -> Tools -> Journal -> Journal Language 選擇存為 Python.
  2. 在NX 上方功能表最右方的 Toolbar Options 選項設定中, 開啟 Menu 表單, 從 Menu -> Tools -> Journal 中可以選擇 Play, Record 或 Edit 功能.
  3. 利用 NXOpen Python 以遞迴方式, 從已經開啟的 uarm 機械手臂組立檔案中, 查詢各零件的物理性質, 列出各零件的體積後, 計算各零件的重量
